# STATISTIC_CALCULATOR
PROBABILITY AND STATISCTICS PROJECT 3RD SEM
STATISTIC_CALCULATOR
A Java program to calculate Mean, Median, and Mode for three types of data:

Individual data (raw values)

Discrete data (values with frequency)

Continuous data (class intervals with frequency)

Features
Textbook-style output shows calculation steps

Easy menu navigation and robust error handling

Supports re-calculation and data review

How to Run
Download StatisticsCalculator.java from this repo.

Open your terminal and navigate to the folder containing the file.

Compile the program:

bash
javac StatisticsCalculator.java
Run the calculator:

bash
java StatisticsCalculator
Usage Instructions
You will be prompted to choose a data type:

Individual Data: Raw numbers, enter each value one by one.

Discrete Data: Enter (xi, fi) pairs; xi = data value, fi = frequency.

Continuous Data: Enter intervals as (lower, upper, frequency).

Choose which statistic you want for each type (Mean, Median, or Mode).

Review data after entry and confirm before calculation.

Error messages help you correct invalid input.
